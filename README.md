# Sample .NET MAUI app
### Demonstrating how to create .NET MAUI app with the following components
- .NET MAUI
- .NET MAUI Community Toolkit
  - MVVM pattern
  - Messenger Pub/Sub (Communication between view models)
- SQLite
- Guard Clauses
- Bogus (Data Faker)
